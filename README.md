 Igreja Solidária - PIBVC

 Sobre o Projeto

Sistema de gestão de doações e ações sociais desenvolvido para a Igreja Batista em Vila Candoza (PIBVC) como projeto de extensão da Universidade Iguaçu (UNIG).

 Funcionalidades

- Cadastro de famílias beneficiárias
- Registro de doações (alimentos, roupas, higiene)
- Pedidos de ajuda
- Algoritmo de matching para casar doações com pedidos
- Registro de ações de saúde
- Relatórios de impacto
- Exportação para Excel e CSV

Tecnologias Utilizadas

- Python
- Flask
- SQLite
- HTML/CSS

Como Executar

1. Instale o Python 3.10+
2. Instale as dependências: `pip install flask pyjwt openpyxl`
3. Execute: `python app.py`
4. Acesse no navegador: `http://localhost:5000`
